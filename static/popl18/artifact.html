Examples for the paper

We provide interactive Liquid Haksell code for the examples presented in Sections 2 and 3 of the paper. The Liquid Haskell queries are checked by sending requests to the Liquid Haskell server hosted at http://goto.ucsd.edu:8090/.

  • Section 2: html file for Overview
  • Section 3: html file for Natural Deduction




Install Liquid Haskell

You can install Liquid Haskell using cabal

cabal install liquidhaskell

or from the github source following the installation instructions in the repo.




Run the benchmark suite

To run the benchmark suite there are three options.

  • Install the popl18 repo branch

  • User docker
